Welcome to the Prosim project

Prosim stands for PROduct SIMilarity.

The acronym OFF used in this documentation stands for OpenFoodFacts.

goal

This project aims at aggregating information delivered by the OpenFoodFacts collaborative database in order to deliver more or less similar products towards a reference product of your choice, classified for instance by their nutrition score.

But it could be enhanced by the community to take into account other criteria regarding vegans, additives, allergens, etc.

processes

The Prosim database is built by increment. Feeders are responsible for this task.
  • a first build by intersecting from the official OFF-database all products with each-other
  • An increment build which intersects all products of the Prosim database this time (feeder_1) with new or updated products from the OFF-official database (feeder_2).
